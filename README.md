# Task Manager API

School Project - Laravel Backend

## About

This is my school project where I built a REST API using Laravel for managing users and tasks.

## Tech Stack

- Laravel 10
- MySQL
- PHP 8+

## Features

- User CRUD
- Task CRUD  
- User-Task relationship

## Quick Start

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
